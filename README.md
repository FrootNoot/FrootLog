## About The Project

A web app I created to help me track my workouts, provide insights and to let people see how my progress is going.



### Built With
- **Frontend:** React (Netlify)
- **Backend:** Node.js & Express (Render)
- **Database:** PostgreSQL (Supabase)
- **Data Fetching:** Axios

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

### Installation

1. Get a free API Key at [https://example.com](https://example.com)
2. Clone the repo
   ```sh
   git clone https://github.com/FrootNoot/FrootLog.git
   ```
3. Add .env file into backend/
   ```js
   DATABASE_URL=your_supabase_postgres_url
   PORT=5000
   ```
4. Run backend with
   ```sh
   cd backend
   npm install
   npm start
   ```
5. Add .env file into frontend/
   ```js
   REACT_APP_API_URL=https://your-backend-service.onrender.com
   ```
6. Run Frontend with
   ```sh
    cd frontend
    npm install
    npm start
   ```
